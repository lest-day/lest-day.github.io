> [职员帖文本生成器](https://lest-day.github.io/staff-post-text-generator/text-generator.html)
> ---
> 通过选择处理类型情况并且提供处理者的用户名和被处理者的用户名来生成准确和规范的职员帖文本，在复制后通知被处理者。
